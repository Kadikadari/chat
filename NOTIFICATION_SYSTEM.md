## 🔔 نظام إخطارات الرسائل الخاصة

### الميزات المضافة:

✅ **استقبال الرسائل الخاصة تلقائياً**
- يظهر إشعار (notification) منبثق عند استقبال رسالة خاصة جديدة

✅ **فتح المحادثة تلقائياً**
- عند النقر على "فتح الرسالة" في الإشعار، يتم فتح نافذة المحادثة تلقائياً

✅ **تصميم احترافي**
- إشعار منبثق جميل مع خلفية شفافة
- أنيميشن سلس عند الظهور والاختفاء
- إغلاق تلقائي بعد 8 ثوانٍ

### المكونات المضافة:

#### 1. `useIncomingMessages` Hook
**الموقع:** `lib/hooks/useIncomingMessages.ts`
- يستمع للرسائل الجديدة الموجهة للمستخدم الحالي
- يتحقق من الرسائل غير المقروءة
- يشغل صوت إخطار (اختياري)

```typescript
const { incomingMessages, hasNewMessage, clearNotification } = useIncomingMessages();
```

#### 2. `MessageNotification` Component
**الموقع:** `lib/hooks/components/MessageNotification.tsx`
- مكون عرض الإشعار المنبثق
- يعرض اسم المرسل والرسالة
- يوفر زر "فتح الرسالة" و "إغلاق"

#### 3. تحديثات `RoomsPage`
**الموقع:** `app/rooms/page.tsx`
- إضافة الـ hook للاستماع للرسائل الجديدة
- عرض الإشعار عند وصول رسالة جديدة
- معالج فتح المحادثة الخاصة تلقائياً

### طريقة العمل:

1. **عند الدخول لصفحة الغرف**: يبدأ تطبيق `useIncomingMessages` بالاستماع للرسائل الجديدة
2. **عند استقبال رسالة خاصة**: يظهر إشعار منبثق فوراً
3. **اختيار المستخدم**:
   - النقر "فتح الرسالة" → فتح المحادثة الخاصة
   - النقر "إغلاق" أو انتظار 8 ثوانٍ → إغلاق الإشعار

### التخصيص:

يمكن تعديل السلوك من خلال تعديل الثوابت في الملفات:

**مدة الإغلاق التلقائي** (في `MessageNotification.tsx`):
```typescript
const timer = setTimeout(() => {
  handleClose();
}, 8000); // 8 ثوانٍ - غير هذا الرقم حسب الحاجة
```

**صوت الإخطار** (في `useIncomingMessages.ts`):
```typescript
const audio = new Audio("/notification.mp3"); // تأكد من وجود الملف
audio.play().catch((e) => console.log("لا يمكن تشغيل الصوت:", e));
```

### ملاحظات مهمة:

⚠️ **تأكد من إضافة ملف الصوت**:
- أضف ملف `notification.mp3` أو `notification.mp3` في المجلد `public/`
- أو عدّل المسار في `useIncomingMessages.ts`

✅ **اختبار الميزة**:
1. افتح المتصفح في نافذتين
2. سجل دخول بحسابين مختلفين
3. أرسل رسالة خاصة من حساب لآخر
4. يجب أن يظهر إشعار منبثق تلقائياً
